# Casa · House Designer

A browser-based house design tool. Draw your floor plan on a Sims-style dot grid
(walls, rooms, doors, windows, floors, furniture), then press **View in 3D** to see a
live extruded 3D model. Recolor walls / floors / furniture, stack multiple floors, and
load starter templates. Saves to your browser automatically; export/import as JSON.

Single self-contained `index.html` (Three.js from CDN).
